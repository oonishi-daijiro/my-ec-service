# my-ec-service
これはプログラミング３の授業で作成したECサイトのデモです．

## How To Run
```
git clone https://github.com/oonishi-daijiro/my-ec-service && cd my-ec-service
```
```
npm i
```
```
npm run dev
```
```
see http://localhost:3000
```
